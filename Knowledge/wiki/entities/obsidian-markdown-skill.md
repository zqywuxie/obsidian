# obsidian-markdown Skill

> 让 LLM 能正确编写 Obsidian Flavored Markdown 的技能。已安装到 `.claude/skills/obsidian-markdown/`。

## 功能

- [[Wikilink]] 内部链接（含 heading/block 级别）
- `![[Embed]]` 嵌入（图片、PDF、其他笔记、音视频）
- `> [!callout]` 高亮块（note/tip/warning/info/example 等）
- YAML frontmatter 属性（tags, aliases, cssclasses）
- 文本高亮 `==text==`、注释 `%%text%%`
- Mermaid 图表、LaTeX 数学
- 脚注、任务列表

## 在本 Wiki 中的使用

本知识库的所有 `.md` 页面编写均依赖此技能。[[CLAUDE.md]] 中定义的页面规范与此技能一致。
